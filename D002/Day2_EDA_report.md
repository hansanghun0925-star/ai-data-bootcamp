# 따릉이 대여 데이터 — 기초 EDA 리포트 (여기에 작성)

## 1. 데이터 개요
- 행/열: ( 행,  열)
- 기간: 2025-01-01 ~
- 주요 컬럼: date, rentals, avg_temp, rainfall, station_area

## 2. 구조 진단 (shape / info / describe)
- 자료형 요약: date 열은 datetime64으로 인식됨. rentals 열은 float64, avg_temp 열은 float64, rainfall 열은 int64, station_area 열은 str 문자열로 인식됨.

- 수치 요약에서 눈에 띈 점: 수치 요약에서 avg_temp의 최댓값이 250. 평균이 14이고, 기온이라는 특수성을 고려했을때 이상치 의심.
- rentals 평균 4531. 
- rentals 열에서 8개의 결측치 확인 (count 개수 타 열과 불일치)/ 건수가 생긱보다 많아서 놀랐다. 

## 3. 품질 진단 (결측 / 중복 / 표기)
- 결측: rentals 8건 외 존재 x
- 중복: 1건 [2025-01-04 3295.0 0.9 0 강남]
- 표기 혼재: station_area에 ' 강남 '(공백) 5건 존재

## 4. 패턴 (시각화에서 읽은 것)
- 추이: 본격적인 겨울철인 1~2월의 대여량(rentals)은 전체 평균인 4,531건을 밑돌았으며, 특히 1월에는 연중 최저치에 근접하는 양상을 보였습니다. 이후 날씨가 풀리는 3월에 접어들며 대여량이 반등하기 시작해 가파른 상승세를 탔고, 4월에 연중 정점을 찍었습니다. 하지만 5월부터는 다소 감소하는 흐름이 나타났으며, 초여름인 6월에는 다시 평균치 아래로 떨어지다가 기온이 높아지는 7월에 최종적으로 최저점을 기록했습니다.
- 관계:기온과 rentals 수는 활동하기 좋은 기온인 10-15'c를 축으로하는 이차함수 형태를 가지고 있습니다. 이에 기온과 rentals 수가 양의 상관관계, 선형적인 분포를 보인다고는 말할 수 없으며, 15도가 넘어가는 시점을 변곡점으로 하는 역U자형 비선형관계를 갖고 있다고 말할 수 있습니다. 

## 5. 다음 분석 제안
- 추가적으로 기온과 대여량(rentals)의 관계를 보다 심층적으로 파악하기 위해, 대여량이 꺾이는 임계점(15℃ 부근)을 기준으로 데이터를 분할하여 각각의 선형적(Linear) 추이를 분석해 보고자 합니다.
기온 변화에 따른 대여량의 기울기가 가파를수록 사람들의 야외 활동 심리가 민감하게 반응한다고 해석할 수 있습니다. 따라서 이러한 기울기(민감도) 분석은 향후 효과적인 광고 집행 시점이나 타겟 프로모션 기간을 선정하는 데 유의미한 지표로 활용될 수 있을 것으로 사료됩니다.
